# Changelog

## [1.0.0] - 07-04-2026
### NO LONGER IN ALPHA VERSION!

## [0.2.2-alpha] - 06-04-2026
### Added
- Google Docs, Notion, Google Drive and Youtube to Quick Actions
### Changed
- Goals, Reminders and Tasks can now be removed if completed
- Updated the fallback prompts (if API for the AI fails)
- Can now multi-select for Reminders, Goals and Tasks for deletion
### Removed
- Removed background music

## [0.2.1-alpha] - 05-04-2026
## Added
- Goals are now available
### Changed
- Tasks are now updated with much more specific

## [0.2.0-alpha] - 05-04-2026
### Added
- Now features AI using Open-source model. (Using OpenRouter's API)
- Added Fallback to the old use of command lists, in case of AI's API decided to fail

## [0.1.3-alpha] - 05-04-2026
### Changed
- Code for Alarm sound now uses st.iframe due to removal of st.components.v1.html

### Changed
- Reminder now updated to add an alarm system
- Music now becomes playlist music, can upload multiple .mp3 files

## [0.1.1-alpha] - 04-04-2026
### Added
- Can now edit and delete notes
- Added background music, upload your .mp3 file and it'll play in the background for you

## [0.1.0-alpha] - 04-04-2026
### Added
- Initial Streamlit app
- Basic reminder system with st.toast()
